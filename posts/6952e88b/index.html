注意：要使这些规则生效，被忽略的文件或目录需要以 /* 结尾。此外，请参考以下规则写法：

  • 以斜杠 / 开头表示目录；
  • 星号 * 可匹配多个字符；
  • 问号 ? 匹配单个字符；
  • 方括号 [] 内包含单个字符的匹配列表；
  • 大叹号 ! 表示不忽略（跟踪）匹配到的文件或目录；

你可以使用 git update-index --assume-unchanged <file> 命令来取消对一个文件的跟踪。这适用于你希望暂时停止 Git 监控特定文件变动的情况。
