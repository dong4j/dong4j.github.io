exists 和 missing 过滤

match_all 查询

使用 match_all 可以查询到所有文档，是没有查询条件下的默认语句。

\n\n\n

此查询常用于合并过滤条件。 比如说你需要检索所有的邮箱, 所有的文档相关性都是相同的，所以得到的 _score 为 1
